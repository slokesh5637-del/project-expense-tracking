# 💰 Expense Management System

A full-stack **Expense Management System** built using **Streamlit (Frontend)** and **FastAPI (Backend)**.  
This application helps users track daily expenses, analyze spending patterns, and manage finances efficiently.

---

## 🚀 Technologies Used

- Python  
- Streamlit (Frontend)  
- FastAPI (Backend)  
- SQL (Database)  
- Postman (API Testing)  
- Git (Version Control)

## ⚙️ Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/slokesh5637-del/project-expense-tracking.git
cd project-expense-tracking
2. Install Dependencies
pip install -r requirements.txt
3. Run Backend
cd backend
uvicorn server:app --reload
4. Run Frontend
cd frontend
streamlit run app.py
🧩 Problem Statement

Managing expenses manually becomes complex and unorganized.
This system provides a structured way to:

Add & update expenses
Track category-wise spending
Analyze monthly trends
Make better financial decisions
🖥️ Project Overview
🔹 Add / Update Expenses

Users can add or update expenses by selecting date, amount, category, and notes.

🔹 Analytics by Category

Shows expense distribution across categories.

🔹 Analytics by Month

Displays monthly expense trends.
